다음 중 다항식 5xy - y2 + 2x2 + 3y + 1를 x에 대한 내림차순으로 정리한 것을 골라라.
  • 1

    1 + 5yx + 2x2 - y2 + 3y

  • 2

    1 + 5yx + 2x2 - y2 + 3y

정답풀이 입문

x에 대한 내림차순으로 정리하려면 x의 차수가 높은 항이 앞에 위치하도록 정리해야 한다. ( i ) x2 이 포함된 항은 2x2 이다. ( ii ) x가 포함된 항은 5xy = 5yx 이다. ( iii ) 상수항은 -y2 + 3y + 1 이다. 따라서 주어진 식을 x에 대한 내림차순으로 정리하면 2x2 + 5yx - y2 + 3y + 1 이다.
